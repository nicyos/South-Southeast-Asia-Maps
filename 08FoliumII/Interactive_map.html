Writing GeoDataframe as a shapefile¶

This is not necessary for this project, but if you want to write a GeoDataframe as a shapefile, below is how to do it. Let's use another dataset for all groups (terminated and not terminated), and call it groups_df.

First, zip the longitude and latitude into one column called "geometry".

In [17]:
from shapely.geometry import Point

groups_df = pd.read_excel("AOSA_map.xls")
geometry = [Point(xy) for xy in zip(groups_df.Longitude, groups_df.Latitude)]
groups_coords = gpd.GeoDataFrame(groups_df, crs = result.crs, geometry=geometry)

groups_coords['geometry'] = groups_coords.apply(lambda x: Point((float(x.Longitude), float(x.Latitude))), axis=1)

Mapping¶

Using folium, make a basemap that starts at 24, 84 for latlong and start at zoom level 5. Ask folium to create a choropleth map on the field "Polity_score", using the Yellow Green color scheme. Caption the legend "Polity Score Scale". Read the terminated groups and non-terminated groups dataframes, and using for-loops, add as markers to the map.

In [ ]:
import folium
from folium.element import IFrame
from geopy.geocoders import Nominatim


m = folium.Map([24, 84], 
               zoom_start=5)

folium.TileLayer('Mapbox Bright').add_to(m)

ft = "Polity_score"
cmap = folium.colormap.linear.YlGn.to_step(9).scale(polity_merge[ft].min(), polity_merge[ft].max())

folium.GeoJson(polity_merge,
               style_function=lambda feature: {
                'fillColor': cmap(feature['properties'][ft]),
                'fillOpacity' : 0.95,
                'weight' : 1, 'color' : 'black'
               }).add_to(m)

cmap.caption = 'Polity Score Scale'
cmap.add_to(m)

folium.GeoJson(result_country, 
               style_function=lambda feature: {'weight': 2, 'color':'black'}).add_to(m)

terminated_groups = terminated
terminated_groups["fullname"] = terminated_groups["fullname"].astype(str)

not_terminated_groups = not_terminated
not_terminated_groups["fullname"] = not_terminated_groups["fullname"].astype(str)

                        + ", terminated in " + str(row['obsyear'])
                        + " with " + str(row['terminateform'])
